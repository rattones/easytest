# teste para a Easy

> por: Marcelo Ratton

### backend em PHP 7.2

o backend foi desenvolvido no servidor embutido no PHP, também funciona em servidor Apache (mas é necessário criar um arquivo .htaccesse não presente neste respositório)

```bash
    php -S localhost:8090
```

### teste da API usando o Insomnia

segue também neste repositório o arquivo `Insomnia.json` com os testes realizados para a criação deste teste

### base de dados

para o teste foi utilizado o PDO com a base de dados em SQLite 3, para facilitar a replicação do teste em qualquer ambiente

### framework PHP

o framework PHP utilizado no projeto foi desenvolvido por mim para estudos e testes como este, ele pode ser encontrado no repositório dele aqui no GitHub
